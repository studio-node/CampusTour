
# Base Prompt

This project is for a Campus tour app and supporting services that I am building for my university, but it will be for many schools. I am in the middle of building it at the moment, and I need your help. In the mobile app portion of this project, there are three user journeys: self-guided, ambassador-led, and ambassador. Ambassadors lead a tour and the ambassador-led tour group can see the changes made by the ambassador. You can find all the overarching documentation for this project in @user-journeys and the whole SupaBase schema and configuration in @supabase_schema.sql . When you create your to-do list, we work one at a time. You do not move on from an item until I have tested it and passed it off. Only then can you move on to the next item on the list.


The second part is bigger. I want the ambassador to be able to push location media to group members. This media should be accessed by the ambassador in a button at the top of  each location's details screen. This button should take them to a screen with a list of all the media related to that location (which is in the `location_media` table). Each row in that list should include a little thumbnail, a name for that media, what type of media it is, and two buttons for pushing the media to the group. The first button will just add the media to the users' detail screens for that location at the bottom. The second button will push the media to users and it will take over their screens so they will all see it, but there should be an 'X' so they can close out if they want. For now, we just want to handle video and images.
